## 프로젝트 개요

### 프로젝트 설명
이 프로젝트는 사용자가 입력한 이름과 발급일자를 기반으로 개인화된 수료증 PDF를 생성하고 다운로드할 수 있는 React 기반 웹 애플리케이션입니다.

### 주요 기능
- 참가자 이름 및 발급일자 입력 폼
- 동적 PDF 수료증 생성
- 자동 PDF 다운로드 기능
- 사전 작성된 수료증 내용 (제목, 과정명, 기관 정보)
- 입력된 이름과 날짜를 수료증에 자동 삽입

### 프로젝트 목적
사용자가 간편하게 개인화된 수료증을 생성하고 다운로드할 수 있는 자동화된 솔루션 제공

## 프로젝트 구성 방식

- React와 Vite를 활용한 빠른 개발 환경 구축
- jsPDF 라이브러리를 이용한 클라이언트 측 PDF 생성
- 폼 입력값 검증 및 동적 PDF 템플릿 처리
- 서버 부하 없이 클라이언트 측에서 PDF 생성

## 프로젝트 실행 방법

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 시작
```bash
npm run dev
```

### 3. 애플리케이션 접속
상단 메뉴에서 'Container → Execution URL and Port → Registered URL and Port → 선택된 행의 단축 버튼 클릭'을 통해 애플리케이션에 접속합니다.

또는 브라우저에서 `http://localhost:3000`으로 접속합니다.

### 4. 수료증 생성
- 폼에 참가자 이름 입력
- 발급일자 선택
- "수료증 다운로드" 버튼 클릭
- PDF 파일이 자동으로 다운로드됨

## 잠재적 오류 및 해결 방법

| 오류 | 원인 | 해결 방법 |
|------|------|---------|
| `npm install` 실패 | Node.js 버전 호환성 문제 | Node.js 20.18.3 버전 확인 및 재설치 |
| 포트 3000 이미 사용 중 | 다른 프로세스가 포트 점유 | `lsof -i :3000` 명령으로 프로세스 확인 후 종료 |
| PDF 다운로드 안 됨 | 브라우저 팝업 차단 설정 | 브라우저 팝업 차단 해제 |
| 한글 폰트 깨짐 | jsPDF 한글 폰트 미지원 | 기본 폰트 또는 웹폰트 설정 확인 |
